## Debugging
```
breakpoint()
```

|command | description |
| --- | --- |
| n | next |
| c | continue |
| l | list |
| ll | long list |
| q | quit | 
| p [A], [B], [C] | print |
| j [LINE] | jump to line |
| step | go inside of function |
| tbreak [LINE] | temporal break |
